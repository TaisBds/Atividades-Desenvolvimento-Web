# 🌐 Atividades de Desenvolvimento Web

Este repositório reúne atividades desenvolvidas com foco em **Desenvolvimento Web**, utilizando HTML e JavaScript.

O objetivo é praticar a criação de páginas interativas e consolidar conhecimentos fundamentais do front-end.

---

## 📂 Conteúdo do Repositório

- `index.html` → Estrutura principal da página  
- `script.js` → Lógica e interatividade com JavaScript  

---

## 🛠️ Tecnologias Utilizadas

- 🌐 HTML5  
- 🟨 JavaScript  

---

## 🧠 Conceitos Trabalhados

Durante o desenvolvimento, foram aplicados:

- ✔️ Estruturação de páginas com HTML  
- ✔️ Manipulação do DOM  
- ✔️ Interatividade com JavaScript  
- ✔️ Lógica de programação
